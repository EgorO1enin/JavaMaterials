Функции хэш таблиц java:

1) put(key, element) - добавить элемент 
2) get(key) - получить элемент по ключу 
3) values() - все элементы таблицы 
4) keySet() - для получения всех ключей 
5) remove(key) - удалить элемент по ключу 
6) clear() - очистить таблицу 
7) containsKey(key) (англ. «содержит ключ») — проверяет наличие ключа  
8) containsValue(value) (англ. «содержит значение») — проверяет наличие значения.
