Работа с JAR файлами

1) Первый этап — скомпилировать исходный код в байт-код. В этом поможет компилятор Java, который реализован в утилите javac

Чтобы не компилировать каждый класс по отдельности, перейдите в папку src и используйте команду javac -d <имя папки> *.java. Опция -d <имя папки> (от англ. directory — «папка, каталог») соберёт все сгенерированные class-файлы в одну папку. В нашем примере это папка с именем bin (от англ. binary — «двоичный», так как байт-код хранится в двоичном формате), которая будет создана автоматически. А *.java означает, что скомпилированы будут все файлы с расширением .java, которые находятся в текущей папке — src.

Команда: javac -d bin -encoding UTF-8 *.java

2) После этого можно создать JAR:

jar cfe <имя jar-файла> <имя стартового класса> <список файлов>

jar cfe library.jar Practicum -C bin .
# Точка в конце строки указывает на то, что в JAR должны попасть 
# все файлы из папки bin.

3) Теперь, когда у вас есть jar-файл library.jar, вы можете быстро запустить программу. Введите в терминале команду java -jar <имя jar-файла>:

java -jar library.jar
