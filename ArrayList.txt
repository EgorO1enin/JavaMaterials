Функции списков java:

1) add(element) - добавить значение в список 
2) get(index) - получить значение элемента по индексу 
3) size() - размер списка 
4) remove(index) - удалить элемент по индексу 
5) remove(element) - удалить элемент по элементу 
6) isEmpty() - проверка на пустоту элемента 
7) contains(element) - проверка содержится ли такой элемент в списке 
8) clear() - очистить список
